Version 7.3

Changes

  • Removed flinching in 1v1. This is a test change to attempt to balance the mode better. I will be carefully observing its effects on the 1v1 matchups.

  • Fixed a bug where charged Chameleon Sting didn't protect against Gallery spikes.

  • Fixed a bug where players could pick up flag while invisible.

  • If a player joins a match with 10 players and some are bots, a bot is now removed to keep the player count at 10.

  • Removed the double tap to dash feature since this was causing dash lockups sometimes. This feature will be fixed and re-added in the next major update.
