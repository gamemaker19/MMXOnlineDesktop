Version 19.1

Asset Changes? Yes

Relay Server Update? No

Bug Fixes

  • Fixed a bug where opening Vile loadout in the main menu would crash the game.
  • Fixed various other menu and controller crashes.
  • Fixed a bug where Unlimited Potential buster could not be shot if buster was equipped in slot 2 or 3.
  • If an visualmods override sprite is no longer in the official sprite folder, it will no longer crash the game on load but simply ignore the sprite.
  • Fixed improper key displayed when showing prompt to revive from Vile 1 to Vile V. It would improperly show the menu secondary key, whereas the right key is the ATTACK/SHOOT button
  • Fixed a bug where Kaiser Sigma could revert to normal Sigma, which caused many other bugs in turn.
  • Fixed a bug where Crush Crawfish could get stuck in hurt/death animation.
  • Added a taunt to Volt Catfish, preventing a softlock if trying to taunt as Volt Catfish.
  • Fixed a bug where Triad Thunder would mini-flinch more than once per triangle.
  • Fixed a bug where Raijingeki could take health and energy pickups.
  • Fixed a bug where players stopped by Dark Hold could still be slowly moved.
  • Fixed a bug where X1 body part was showing X2 body part (due to incorrect spritesheet in assets).
  • Fixed shader issues with Nightmare Zero and Highway 2.
  • Fixed various color issues with the X3 Maverick spritesheet.

Balance Changes

  • There was an undocumented change in v19.0 for parasitic bomb that was forgotten to be mentioned in the v19.0 patch notes, so it will be mentioned here: latched parasitic bomb slows down enemies now.
  • Kaiser Sigma:
    • Increased stomp height requirement and made damage scaling require a greater fall height.
    • Increased landing lag from 0.75 to 1s.
    • Reduced vertical hover speed by half.
