Version 15.3

Asset Changes? No

Relay Server Update? No

Changes/Bug Fixes

  • Fixed a bug where command grabs would leave victims in grab state even if cancelled.
  • AFK timer no longer runs when spectating.
  • Fixed a bug where Quake Blazer did 4 damage instead of 2.
  • Fixed a bug where Flame Mammoth oil fires could desync and become invisible on the defender's screen.
  • Low health mash requirement further reduced. Was 5x, now 3x.
    • 3x means that at 1/16 health, requires 3x the inputs than at 16/16 health.
    • This may continue to be reduced in the next coming patches until it feels right.

Balance changes

All

  • Electric attacks no longer pierce defense; will now only pierce Armored Armadillo's armor.

    • Mega Man X

  • Ultimate Armor: increased cooldown of Giga Buster Plasma Shot from 0.25 to 0.5 seconds.

    • Sigma

  • Guard: reverted undocumented changed causing projectiles to be reflected.
  • Summoner: decreased Maverick cost from 4 to 3 scrap.
  • Tag Team: increased switch cooldown from 2 to 4 seconds.
  • Spark Mandrill: Electric Spark range reduced by 33%.
