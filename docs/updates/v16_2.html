Version 16.2

Asset Changes? No

Relay Server Update? No

Bug fixes

  • Removed X preferred armor option as it's no longer used. This fixes a crash in the character select menu if this option was set to X2 or X3.
  • Fixed a bug where Sting Chameleon couldn't climb all the way up ladders.
  • Fixed a bug in the Wait Menu where the map name was displaying the match name.
  • Fixed command grabs not working offline.
  • Fixed a bug where Chill Penguin could not break ice statues with taunt.
  • Fixed a bug where Rekkoha pillars and Wolf Sigma in Large Camera mode would not spawn at the right Y position.
  • Fixed a cooldown exploit with Nova Strike.
  • Fixed a bug where neutral Ride Armors would recover all HP when changing owners.
  • Fixed a bug where Giga Crush/Nova Strike directional inputs could allow X to shoot weapons in the death animation.

Balance changes

General

  • Giga Attack ammo gain now proportional to max health. The greater the max health, the less giga ammo granted on damage taken.
  • Reworked scrap awarding system:
    • Now works based on multiplicative modifiers. The base scrap earned is dependent on how far behind you are and can be 1-3 scrap per kill (as with before).
    • Killing a Maverick increases the base modifier by 2x (as with before).
    • Killing a player with 4 or more heart tanks (24 or more life bars) now increases the base modifier by 2x.
    • Killing a 5 scrap Hyper Mode (Golden X, MK-II, etc) now increases the base modifier by 2x.
    • Killing a 10 scrap Hyper Mode (UP/UAX, Black Zero, etc) now increases the base modifier by 3x.
    • These modifiers can stack. For example, if heavily behind and you kill an 8 heart tank Black Zero, the raw scrap gained is 3x2x3=18 scrap.
    • Remember, the existing 15 scrap limit (which prevents hoarding) still applies. If over 15 scrap you will always earn 1 scrap per kill. If under 15 scrap you can only gain enough to go up to 15.

Zero

  • Rekkoha ammo usage reverted to 32 bars.

Sigma

  • Summoner: no longer will issue attack move command when dashing, which should help prevent accidental attack moves.
  • Added a new scrap gain ability exclusive to Sigma called "War Economy":
    • Each Maverick in Sigma's loadout not yet purchased and deployed to combat will fund Sigma's war efforts behind the scenes, providing 1 scrap every 10 seconds.
    • This ability does not apply to Striker Mode since Mavericks in this mode are always considered "deployed".
