Version 8.0: The X2 Update

Mega Man X returns with an all new arsenal of weapons and armor from his second SNES title! Unleash these weapons on 10 new large X2 maps, and 2 new medium and small maps. Also in this update comes a whole host of additions for Zero and Axl and general improvements and fixes.

New Additions

  • X gets 8 new special weapons from Mega Man X2.

    • See more info in the X weapon guide: X Weapon Guide

  • X gets 4 new armors and the Shoryuken from Mega Man X2.

    • See more info in the X armor guide: X Armor Guide

  • Added 10 new large maps and 2 new medium maps from Mega Man X2. These include the intro stage, all 8 maverick stages, and the first X-Hunter stage.

  • Added 2 new 1v1 maps: Central Computer (where X fought Zero in MMX2), and Zero Virus (where X fought Zero in MMX5).

  • Added Zero alternate techniques for Ryuenjin (E-Blade), Hyouretsuzan (Rakukojin) and Rakuhoha (Rekkoha).

    • These can be activated by double tapping the appropriate direction and then pressing SPECIAL.
    • The Raijingeki did not get an alternate technique per se, but a weaker and faster purple colored version of the technique can be incorperated in Zero's Z-Saber Combo after the 1st or 2nd Slash, similar to in MMX4.
    • See more info in the Zero character guide: Zero Character Guide

  • Added White Axl hyper mode.

    • Activate by charging the Copy Shot with 10 scrap. The powerup lasts 10 seconds.
    • White Axl gains the Double Bullet, a pair of dual wielded Axl Bullets, which have twice the fire rate.
    • In addition, White Axl recharges ammo much faster and can hover indefinitely until the powerup runs out.
    • See more info in the Axl character guide: Axl Character Guide

  • Axl gets a new weapon: the Ray Gun from MMX7 and 8.

    • This sci-fi handgun rapidly shoots slow-moving, team-colored lasers that can pass through enemies.
    • Hold down the fire button for automatic fire, which can create a continuous beam if stationary.
    • SPECIAL button with the weapon shoots lasers that can't pass through enemies, but bounce off walls and travel further.
    • See more info in the Axl character guide: Axl Character Guide

Improvements/Changes

  • Fix a performance issue that was slowing down framerates, especially for Axl cursor mode.

  • Game now supports framerates lower than 60 without slowing down gameplay. The minimum framerate supported wihout slowdown is 30 FPS. You can also set a limit on the framerate in Settings (min 30, max 60).

  • Host can no longer kick players without a vote in public matches.

  • Players can no longer become spectators in public matches if this would unbalance the teams.

  • Host can no longer remove a bot in public matches if this would unbalance the teams.

  • Disabled autobalance in private matches.

  • Players can now change teams in offline and private matches.

  • Axl analog aim is now an explicit setting. By default it's disabled, which will not allow 360 degree aim with an analog stick, but allows DPAD aim. If enabled, analog uses 360 aim, but DPAD aim could no longer work on controller.

  • Changed the Axl Lock On Sound setting to now enable/disable lock-on altogether for directional and angular modes.

  • Removed the Axl Backwards Aim Invert setting as it was a niche case that wasn't really used or understood.

  • Re-added double tap to dash. This can be disabled in Settings if it gets in the way.

  • Players selecting Axl for the first time now must select an aim mode first.

  • Maps now have a more descriptive display name in all menus except for the Join Server lobby, where it's still abbreviated.

  • All characters now have 32 HP in 1v1.

  • In Training you can now press F12 to alternate the AI between 4 behaviors: attack, jump, fight and idle.

  • Can now set overrides for loop start and end times when replacing music in the music folder. Read/modify the _loopoverrides.ini file in the folder for more details. Currently only .ogg is supported.

  • Made Black Zero's hair color lighter to more closely resemble official art.

Bug Fixes

  • Fixed instances in maps where Vile and other high jump characters could get out of bounds.

  • Fixed some issues in finding servers in Join Server menu as well as joining private matches.

  • Fixed a bug with X that let you upgrade armor with scrap earned from other characters.

Balance Changes

General

  • (X, Axl) Each weapon now has its own shoot cooldown. This makes it possible to switch weapons and immediately fire it, even if the previous weapon was a slow, laggy one like Storm Tornado, enabling weapon switch combo techniques.
  • Sigma1 Map: Increase capture time required by blue team for first Control Point.

X

  • Fire Wave: now can be damaged by uncharged storm tornado (2 damage per hit with flinch) as it's a weakness.
  • All helmet upgrades now protect against Axl headshots.
  • Improved Hadouken input, making it easier to pull off.
  • Hadouken animation now has super armor, like with the newly added Shoryuken.
  • Reduced charged rolling shield contact damage from 2 to 1.

Zero

  • Ryuenjin no longer flinches. This property has instead been given to the E-Blade, the alt technique for the move.
  • Hyouretsuzan has a new animation with a longer landing lag and smaller hitboxes.
  • Crouch slash has a new animation and can no longer be cancelled into the standard 3 slash combo.
  • Guard damage reduction reduced from 2x to 1.5x.
  • Axl can no longer headshot Black Zero.

Vile

  • Summoning your mech now has a 4 second cooldown between recalls.

Axl

  • Transformations no longer cost scrap, nor heal Axl on activation, though they still replenish his ammo.
  • Expanded invuln frames of Dodge Roll to all frames, and made the roll last a little longer.
  • Blast launcher no longer can be fired when ammo is less than the required amount.
