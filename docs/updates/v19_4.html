Version 19.4

Asset Changes? No

Relay Server Update? No

Bug Fixes

  • Fixed Axl weapon switch cooldown exploit.
  • Prevented Buster Zero from swinging saber while in respawn frames, and other states where it could be improperly used.
  • Fixed bug where Black Buster Zero could not shoot double buster combo with 0 scrap.

Balance Changes

  • Reduced Infinity Gig range by 1/3.
  • Increased Crush Crawfish's claw attack hit cooldown from 0.5s to 1s.
  • Increased Axl weapon switch cooldown from 0.25s to 0.3s (0.5s to 0.6s for low fire rate weapons).
  • Rumbling Bang can no longer assist.
  • An Awakened Zero nerf from 19.3 was undocumented, so mentioning it here: the base saber swing + beam now each deal 3 damage only. This was due to the same attack being shared with Buster Zero. However I am leaving it like this as the old Awakened Zero was deemed by many as dealing too much damage.
