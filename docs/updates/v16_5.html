Version 16.5

Asset Changes? No

Relay Server Update? No

Bug Fixes

  • Fixed a bug/exploit where X could Hadouken or Shoryuken mid-air.
  • Fixed a bug/exploit where activating Wolf Sigma could continually generate kills/deaths if activated on top of kill zones, continually feeding scrap and points to the killer.
  • Fixed a bug/exploit where Wolf Sigma could become invincible and linger around if quickly switching characters or to spectate in the animation.
  • Fixed a bug/exploit where Awakened Zero could rapidly shoot sword beams in certain states.
  • Fixed an issue with projectiles not being cleaned up and destroyed properly which could slow performance in long matches.

Balance changes

Mega Man X

  • Hadouken and Shoryuken: Now requires 3 scrap per use, unless all Heart Tanks and Sub Tanks have been purchased, in which case will not require scrap.

Vile

  • Vile MK-II: Grab can now heal again.
  • Vile MK-II: Back Shield defense only reduces damage by 1 now, and no longer protects against flinch.
