Asset Changes? Yes

General Changes

Improvements

  • Map thumbnails shown in create next match.
  • Increased name limit to 10. Symbolically, this allows "Mega Man X" to be used as a name. It won't increase any more.

Bug Fixes

  • Fixed broken Quake Blazer explosion sprite.
  • "Blocked" text in koth/cp was too high.
  • Flame mammoth fires, magnet mines, and frost shield now move with moving platforms.
  • Fixed softlock on 1v1 end screen.
  • "Please select a map" prompt was not blocking input.
  • Fixed a bug where a 1v1 Maverick could be selected in non-1v1, resulting in a softlock.
  • Fixed destructable wall spawning too many gibs in netcode.
  • Fixed Sub Tank sound improperly playing when hitting Mavericks with healing projectiles.
  • Fixed healing sound not playing when hitting mavericks, Ride Armors with healing projectiles.
  • The "cannot use subtank" message no longer overlaps with Sigma's Maverick icons.
  • Fixed a visual issue with "client and server custom map checksums do not match" menu.
  • Fixed hitbox bugs in crouch states.

Mega Man X Changes

Bug Fixes

  • Magnet Mines can no longer be controlled when dead.
  • Fixed bug where Ultimate Armor could use subtanks when invisible in C-Sting.

Balance

  • Unlimited Potential rework:
    • Buster now flinches and damage increased to 3, but rate of fire greatly reduced, and charge rate reduced to 1x.
    • Grabbing, Parrying no longer eats up drain meter/drain time.
    • Special Saber replaced with Punch.

Vile Changes

Bug Fixes

  • Fixed a bug/exploit where Ride Armor could land and punch at the same time.

Axl Changes

Improvements

  • Improved Ice Gattling sprites.

Bug Fixes

  • Fixed a bug where Gaea shield was not blocking projectiles.

Sigma Changes

Improvements

  • New "Command Button" key introduced for Sigma, which is now used for issuing commands to Mavericks, that can be bound in Sigma's controls menu. This defaults to CONTROL on keyboard for new users. On controller, it is by default unbound and must be bound in Sigma's controls menu. It can be bound to the same as SPECIAL if you want the old Sigma command button behavior.
  • Mavericks can now drop down ladders by pressing DOWN.
  • Improved walk animations of Storm Eagle and Armored Armadillo.

Bug Fixes

  • Fixed a bug where Wire Sponge's up strike chain could get stuck in moving platforms, getting Wire Sponge stuck in the up shoot state.
  • Fixed a bug where Wire Sponge could cling in midair if clinging to a moving platform.
  • Fixed a bug where Commander and Neo forms could not attack in air if Zero swap air input setting was on.
  • Fixed a bug where unarmored Armored Armadillo roll was using the wrong hitbox.

Balance

  • Summoner/Puppeteer/Striker Maverick HP reduced from 32 to 24.
