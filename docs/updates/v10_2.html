Version 10.2

Changes

  • Hyper Buster can no longer be used with the flag.
  • X can no longer headbutt enemies when invisible.
  • Prevented X from unequipping Gold Armor which could permanently give him 3 chips.
  • Fixed a bug where switching to Blast Launcher was taking a full cooldown cycle for Explosion rather than the intended 0.25 seconds.
  • Fixed a bug where Quarry could not be played on CTF.
  • Fixed a bug with charged Gravity Well not crediting on environmental kills.
  • Fixed a visual bug where Vile MK-II could glow after being revived.
