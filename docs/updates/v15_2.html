Version 15.2

Asset Changes? Yes

Relay Server Update? No

Changes/Bug Fixes

  • Increased default Favor the Attacker ping limit to 200.
  • Netcode model and ping limit settings now persists across matches.
  • 1v1 mode: can now change netcode model to Favor the Attacker (however, 1v1 mode will still default to Favor the Defender.)
  • Fixed issues with several command grabs, such as Dead Lift, Energy Drain not working online.
  • Command grab hit registration was made more lenient in Favor the Defender.
  • Fixed a bug in Favor the Defender where projectiles could linger in bugged visual states after being destroyed.
  • Ray Gun: Charge Beam now always favors the attacker.
