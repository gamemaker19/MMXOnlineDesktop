Version 17.2

Asset Changes? Yes

Relay Server Update? No

Bug Fixes

  • Fixed a bug where using Sub Tanks as Sigma could sometimes crash the game.
  • Fixed a bug where charged Chameleon Sting could result in flashing black screens for users using optimized sprites.
  • Fixed a bug where Revolver Barrel and Ancient Gun projectiles were both FTA and FTD, resulting in excessive hit registration.
  • Add a "ceiling" flag to maps. Hyoroga Zero can't cling on to them.
  • Fixed a bug where Zero could use F-Splasher with flag.
  • Fixed collision issues with level 2 buster projectile.
  • Fixed a bug where region chosen at first game launch could not be respected.
  • Fixed a bug where Vile could shoot both cannon and cutter at once
  • Fixed a Highway collision issue.
  • Fixed muzzle position on Ancient Gun.

Balance Changes

  • Reduced Double Bullet and Ancient Gun fire rate from 2x to 1.5x.
  • Increased Mettaur Crash and Ancient Gun range.
  • Ancient Gun bullets no longer disappear on headshot.
  • Ancient Gun scrap cost reduced from 15 to 10.
