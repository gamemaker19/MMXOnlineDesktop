Version 14.1

Asset Changes? No

Bug fixes

  • Fixed a bug where dropping the flag could softlock the match for everyone.
  • Fixed a bug where Wolf Sigma's head could be moved away from its anchor position.

Balance changes

Mega Man X

  • X6 Saber now has 0.75s cooldown between swings.
  • Head Chip now heals a maximum of 32 bars per life.

Misc changes

  • I am reworking the way downloads work. From this point on, the 32-bit SC (self contained) version will be the "default" download that all new players will be encouraged to download as the most convenient option. This version should universally work on all PCs that support the game. The other 3 versions will still be provided for advanced users (32-bit non-SC, 64-bit SC, 64-bit non-SC) but they will not come with an assets folder. If you want/need to use these versions, you'll have to manually download the assets folder, now available as a separate download on the home page, and put it in your game folder. Every change log from now on will indicate whether an assets change was made at the top, so use this to determine when you need to re-download. Another way to check is if the version number of the assets download label on the home page changed from last time.
    • While this seems like more work, it helps save storage space and build time on my end, for a game whose file size is rapidly growing with each release.
    • Even for end users, while it seems like more work to download and install, it would reduce the amount of bytes needed to be download small updates.
  • Corrected documentation in 14.0 changelog that mentioned that the ultimate armor can only be used once per match. This is incorrect and came from a scrapped balance concept for the armor.
