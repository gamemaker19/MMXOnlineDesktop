Version 7.1

Changes

  • Fixed a bug where Axl bombs could push players indefinitely left or right.

  • Fixed a bug where spectators couldn't cycle to the last player in the match.

  • Fixed a bug where Sigma1 stage platforms could get stuck if the host left.

  • Fixed a bug where Zero couldn't pick up ammo capsules.

  • Sigma1: moved blue flag and blue spawns further back.

  • Sigma1: moved red flag position closer to entrance.

  • Sigma1: added an alternate "back door" route into the fortress that can be accessed by climbing the upper wall over the entrance. Currently only Zero and Vile can reach this wall (Axl too if he copies a Zero) but future updates to X should give him this ability, too.
