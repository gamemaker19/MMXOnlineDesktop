Version 14.5

Asset Changes? No

Bug fixes

  • Fixed a bug where Armored Armadillo could continue guarding even after losing armor.
  • Maverick switching will now remember Armored Armadillo armor status as well as Sting Chameleon cloak use count.
  • Fixed a bug where rolling as Axl would release a charging Copy Shot.

Balance changes

General

  • All electric moves in the game now pierce defense buffs and guard moves (including Armored Armadillo guard, back shield, etc).

Mega Man X

  • Charged Storm Tornado: added 1 second flinch cooldown, but decreased damage cooldown from 0.5 to 0.33.
  • Uncharged Magnet Mine: reverted accidental and undocumented cooldown buff.

Vile

  • Rocket Punch projectile spawn point moved closer inside of Vile, making it easier to hit foes that are close by.
  • Distance Needler now pierces defense.
  • MK2 no longer has reduced cooldown on cannons and rocket punchs (missiles still have a substantial cooldown reduction.)

Axl

  • Axl Bullets: added an ammo bar that immediately recharges when not shooting (like Vulcan).
  • Dodge Roll: added an Axl setting that lets you visualize the cooldown via a progress bar.
  • Dodge Roll: first and last frames of the roll animation no longer grant iframes.
  • Spiral Magnum: increased primary fire ammo usage from 2 to 4.
  • Stealth Mode: no longer heals on activation.

Sigma

  • Added a 2-second cooldown for wall dash move from the ground.
  • Summoner/Puppeteer: can now recall all active Mavericks back to base with UP + SPECIAL. This allows Sigma re-summon them at his location of choice, keeping them protected until then.
  • Tag Team: Maverick health reduced from 32 to 16.
  • Armored Armadillo: can now be damaged while rolling, but has 50% defense in this state.
  • Sting Chameleon: can now be damaged by boomerangs (both from X and Boomer Kuwanger) when cloaked.
  • Boomer Kuwanger: can now damage cloaked X's with Boomerang Cutter.
  • Spark Mandrill: can now break X's Rolling Shield with Electric Spark.
  • Velguarder: fire breath no longer works underwater, for consistency with other fire moves.
