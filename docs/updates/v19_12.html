General Changes

  • Official servers have been shut down, and the pre join/create match menu has been updated accordingly. Now, to select the "internet" option and play online, you'll need to edit "region.txt" in your game folder (or MMXOD documents folder if you are using that) and set the ip address to that of a self-hosted server. Otherwise the menu will not let you advance. See the decom guide on how to self-host a server. Then restart the game for it to pick up the change. The region name can be anything but must be set. For quick reference here is the format of region.txt that the game accepts:
    • {
  "name": "YOUR_REGION_NAME",
  "ip": "YOUR_SERVER_IP"
}
    • Note, "region.txt" file with the right format will be generated automatically now if it does not exist, on game startup.

Balance

  • K-Knuckle: second ground punch damage reduced from 3 to 2, flinch reduced to half flinch.
  • K-Knuckle: ground dash attack (spin kick) damage reduced from 3 to 2, and flinch cooldown now shared with second ground punch.
  • K-Knuckle: air neutral kick no longer flinches.
  • K-Knuckle: air drop kick damage reduced from 6 to 4, and flinch reduced to half flinch.
  • Rekkoha: damage per hit reduced from 4 to 3.
