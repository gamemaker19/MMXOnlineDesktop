Version 12.3

Bug Fixes

  • Fixed an exploit where Axl could never run out of ammo for certain weapons.
  • Fixed some instances where Wind Cutter projectiles would not return to Axl when colliding with him.

Balance Changes

Mega Man X

  • Hyper charge ammo usage reduced from 50% to 25%.

Zero

  • Hyouretsuzan freeze is now based on both fall speed and height from the ground. If either condition is sufficient, the attack will freeze.
  • Hyouretsuzan will now cause the ice slow down effect if it does not freeze.

Vile

  • Vile MK-II can now earn scrap.

Axl

  • Ray Gun can no longer go through walls.
  • Increased Ray Gun ammo usage.
  • Reduced the range of Black Arrow projectiles.
  • Wind Cutter can now go through walls.
  • Moving Wheel now slows down and deals less damage and flinch for each player it runs over.
  • Moving Wheel now deals 2 damage and no flinch and disappears on hit on its initial frames (before the spikes come out).
  • Plasma Gun alt fire beam can no longer go through walls.
  • Increased Plasma Gun alt fire beam ammo usage.
  • Plasma Gun alt fire beam range reduced.
  • Ice Gattling base freeze time reduced from 5 to 2 seconds.
