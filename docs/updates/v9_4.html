Version 9.4

Improvements

  • To prevent abuse, a player can now only leave and rejoin a match up to 3 times before being blocked from rejoining the match again for 5 minutes. This restriction does not apply for private matches.
  • Votekicks now persist even if a player leaves the match or changes their name.
  • Axl can now climb walls/ladders with two handed weapons. If Axl slides down a wall or climbs a ladder with a two handed weapon, he will temporarily switch to the Axl Bullet weapon.
  • In 1v1, the "0" number key can now be used to switch to X weapon slot 10 (Giga Crush or Hyper Buster).
  • Can now see damage text applied to Ray Splasher turrets.
  • Added a "GetShaderErrors.bat" file in the game folder that can be run as a wrapper to launch the game to find the exact cause of shader compile errors on startup. If you run this, it will spit out any shader compile errors in a file called "shaderErrors.txt" in the game folder that you can send to GameMaker19 on the Discord to help troubleshoot shader compile issues.

Bug Fixes

  • Fixed a bug that could cause a region to crash if two matches were created at the exact same time.
  • Fixed a bug introduced in 9.0 that caused underwater jumps to become much shorter than they should be.
  • Fixed a bug where Axl gun arm was incorrectly inverted in the wall kick state.
  • Axl no longer loses White Axl charge if Dodge Roll is performed.
  • Fixed a bug where Ride Armor could be recalled slower than usual.
  • Fixed a bug where charged Frost Shield and Tornado Fang attached buster position could get off.
  • Fixed occurrances of getting stuck into ground if falling from too high a height.
  • Fixed a bug where a ducking Vile could clip through ceilings if Gravity Welled.
  • Fixed a bug where sound volume was hard coded to 100% in 1v1's.
  • Fixed a bug where X's Beam Saber charge glow was not syncing to other players.
  • Fixed a bug where charged Spin Wheel wasn't syncing properly to other players in a match.

Balance Changes

  • Zero Buster damage reduced from 4/6/8 to 2/4/6.
  • Hawk and Frog Ride Armors now have a limited fly/swim time. A meter will show when flying/swimming that indicates the amount of fuel left. Fuel will recharge after landing.
  • Hawk Ride Armor max HP increased to 24 bars.
  • Spinning Blade now deals double damage to Ray Splasher turrets as it's the weakness. This means that a single hit from either an uncharged or charged blade will destroy a turret in 1 hit.
