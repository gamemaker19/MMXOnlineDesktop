Version 6.1

Bug Fixes

  • Fixed a bug on Windows 7 where Axl's mouse aim would freeze the game.

  • Fixed a bug where Copy Shot could cause an exception to be thrown for some players.

  • Fixed an exception being thrown when sending Axl Shoot RPC over the network.

Balance Changes

  • Axl: Guns no longer require cursor on opponents for a successful hit. (However headshots still require the cursor on an enemy head.)

    • Commentary: It was way too difficult for players to land hits as Axl, especially on characters such as X who have small hitboxes and few moves that make them stand still. This change makes playing Axl more accessible to most players instead of requiring an unreasonable skill level of aim. However, this does dramatically increase Axl's general power level, so additional changes below will attempt to counteract this.

  • Axl Bullet: This weapon can no longer shoot through walls.

    • Commentary: With the Axl bullets now able to hit anywhere on the screen even without careful aim, this change will help offset the power increase and prevent Axl from potentially being too overpowered. Also, it makes it consistent with the Blast Launcher projectile's behavior.

  • Blast Launcher: Damage reduced from 4 to 3

    • Commentary: It's now much easier to land direct hits with the Blast Launcher with the cursor change. So this offsets the extra power and prevents Axl from being able to nuke enemies from full health to zero health with just 4 shots from a single clip, now requiring 6.
