Version 12.1

Changes

  • Fixed a bug where Rakuhouha could hit players in invincible states.
  • Fixed a bug where Vulcan wouldn't drain energy meter when crouched.
  • Fixed weapon switch cooldown exploit with Axl that could cause a lot of high damage projectiles to be fired in a short period of time.
  • Fixed a bug where using a White Axl DNA Core would not grant the Double Bullets.
  • Fixed a bug where enemy Ride Armor HP in 1v1 mode was reporting incorrect HP values.
  • Scoped Axl screen no longer flashes when hit by a non-flinching move. Instead, a red border is temporarily drawn.
