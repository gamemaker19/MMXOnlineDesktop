Version 11.2

Changes

  • Low-level netcode was changed to be more bandwidth efficient.
  • Fixed a bug where using Grid Mode could cause Vile to get stuck in Ride Armor.
  • Enabled Grid Mode in training.
  • Fixed a bug where auto-scan could reveal invisible players.
  • Fixed a bug with Hadouken cooldown change not applying.
  • Fixed a bug/exploit where Zero could quickly climb a wall by spamming the jump key/double jumps.
  • Vile's napalm cooldown now only starts after the projectile is launched.
  • Fixed a bug where 1v1 timer was always being applied and was broke.
  • Fixed a bug where Vile could not be headshot in Ride Armors in online matches.
  • Fixed a bug where Sigma virus could cover the whole map briefly during timer transitions.
  • Fixed a bug where multiple users of charged Chameleon Sting at once could break the invisibility effect.
  • Reduced "mini-flinch" effect duration of E.Spark and T.Thunder by 33%.
  • Reduced beam damage of uncharged Triad Thunder from 3 to 2.
