Version 14.4

Asset Changes? Yes

Additions/Improvements

  • Added official support for the Developer Console, which has been an undocumented feature for a while now. This can be turned on in the Settings Menu. If enabled, Developer Console commands can be run by pressing the BACKQUOTE key in a match. These can be useful for testing and experimentation, but most commands can't be run in online matches to prevent cheating. See the developer console documentation page for more notes and commands: Developer Console
  • Sigma can now summon Mavericks in 1v1, but can only use Striker command mode.
  • The enemy AI's loadout in Training now matches that of the player, allowing for easier testing.
  • Added a setting to allow full sprite mods (such as adding/removing frames, POIs/hitboxes, etc). To prevent gameplay tampering, if this setting is turned on, you can't play online; only offline bot matches are supported.
  • The "Use Optimized Assets" setting now defaults to "No" unless the user's preset graphics config was set to low or medium.
  • Added a setting, "Matchmaking Timeout", that controls timeout to matchmaking network calls. This defaults to 3 seconds. (Before, the timeout was 20 seconds and not configurable so on a server outage, it would take forever to load matches in Join Server menu.) It can be decreased to reduce the amount of time it takes to fetch the match list during server outages, but if set too low, matchmaking calls may fail even if servers are up.
  • Added additional X/Zero/Axl sprite fixes and improvements provided by Bozo.

Bug fixes

  • Made some improvements in the server code to help prevent random server crashes.
  • Made some improvements in the netcode to help fix lingering i-frame issues on respawn.
  • Fixed a bug where the Mavericks' initial move in Striker/Summon modes was inconsistent if using the ATTACK key.
  • Fixed an exploit where Awakened Zero could quickly launch z-saber projectiles on ladders.
  • Fixed a bug where Assassination couldn't be charged with Sigma disguise.
  • Vile's Vulcans now will only recharge when not in the shoot animation. This fixes an exploit where Vulcan could dish out near-normal amounts of DPS by rapidly pressing SHOOT even when E-empty was empty.
  • Fixed a bug where Back Shield could block Unlimited Potential and Launch Octopus drain attacks.
  • Fixed a bug where Axl could shoot a copy shot in invuln frames.
  • Fixed a bug where Chill Penguin blizzard could push cc-immune hyper modes.
  • Fixed a bug where cloaked Sting Chameleon could die from environmental hazards.
  • Fixed a bug where Gravity Well could pull things it shouldn't be able to pull, such as destructable walls.
  • Fixed a bug where Armored Armadillo would not play explosion animation/sounds if killed with no armor.
  • Fixed a bug where Mavericks could be hurt in the warp out state, causing them to enter a glitched state.
  • Fixed a bug where Double Buster kill feed entry was showing the X lemon shot.
  • Fixed a bug where charged Tornado Fang wouldn't damage cracked walls/frost shields.

Balance changes

Mega Man X

  • Gravity Well pull power further reduced.
  • Gravity Well time reduced to 2 seconds.
  • Unlimited Potential: Can now cancel Dr. Light cutscene shortly (about 2 seconds in) after it starts, by pressing SPECIAL again.

Vile

  • Necro Burst can now damage cracked walls.
  • Buckshot Dance ammo usage increased by 20%.

Sigma

  • Decreased sword swing cooldown from 1.25s to 1s.
  • Increased dash speed by 15%.
  • Wall dash move can now be performed on the ground by pressing SPECIAL when dashing, but from the ground it's slower.
  • Sigma no longer can air-dash after a wall dash, but now has dash speed momentum after a wall dash.
  • Summoner mode: Maverick cost decreased to 4.
  • Puppeteer mode: Maverick cost decreased to 3.
  • Puppeteer mode: Saber cooldown timer now runs even when controlling a Maverick.
  • Puppeteer mode: Sigma now guards when controlling a Maverick.
  • Striker mode: can now only have one Maverick on the field at a time.
  • Storm Eagle: Baby eagles are now destroyable.
  • Storm Eagle: Added a height limit to how high players can fly (matching the same behavior as Hawk).
  • Armored Armadillo: If re-summoned at <50% health, armor will be removed.
