Version 3.0: Team Deathmatch

Update 3.0 has been released! This update adds Team Deathmatch mode, the mighty Hadouken, two new Zero abilities, and a whole host of optimizations, improvements and additions!

New Additions

  • Added Team Deathmatch mode. You can join either Red or Blue team. The team that reaches the kill limit first wins!

    • Note that assists don't count for total team kills but still count for earning scrap as X.
    • There's a new in-game menu option, "Match Options" that lets you switch teams. However you can only switch if the opposing team has less players.
    • Also in Match Options, there's a new option for the host to cancel the match (no contest). This is good for if you accidentally set the wrong match settings, or if too many players left for the kill limit to be reached.
    • In the match lobby, the host can also assign players to specific teams with the 0-9 keyboard keys.

  • X can now use the Hadouken (!) in non-1v1 matches.

    • Controls: QCF (quarter circle forward, i.e. down + down forward + forward) + shoot. (Don't forget the shoot button!)
    • It's a 1-hit kill, so beware!!!
    • However, this weapon has some major limitations: it can only be used when X is fully upgraded, AND the move costs 3 scrap, so use it wisely.
    • Zero can reflect this move, so don't blindly toss it out!

  • Zero gets two new abilities:

    • Shippuuga ("Hurricane Fang"). Special button while dashing. This deals less damage than the normal dash attack and has a smaller hitbox but it will flinch on hit!
    • Kuuenzan ("Air Circling Slash"). Special button in the air. This attack will cover an area all around Zero, but deals less damage than the normal air attack. Consider using underwater where the Ryuenjin is ineffective.

  • Added new large map: sigma2. This map takes place in the beginning of the MMX Sigma 2 stage and supports up to 10 players.

Bug Fixes

  • Fixed a bug where switching to Zero and back to X causes the body armor health upgrade to be lost.

  • Fixed a bug where X could shoot a charged X buster even without ammo by switching to it while charged.

  • Fixed a bug where X could continue to decrement ammo even if it ran out.

  • Fixed a bug with full-screen mode causing the game to hang or crash on startup.

Optimizations/Quality of Life changes

  • Optimized and reduced map image sizes, and map images are now only loaded when you start a match, as opposed to all at once on program launch.

  • Allowed game to be played without shaders. If your system does not support shaders, you will not see sprite effects or weapon palettes.

  • Added warning popups before game starts if your system requirements do not meet required or recommended.

  • Crashes will now show a popup before the program closes that gives the error message technical details.

  • Crashes will now be logged in a folder called "crashlogs". (out.txt still works, but the crashlogs won't get erased on next program launch.)

  • When changing maps at the end of the match, you will have more options to change the kill count.

  • All non-1v1 maps now support 10 players max. (But medium maps have a recommended max of 4 players or it will be hectic.)

  • The "play to" count will now default to an intelligent value that is based on the game mode and number of players in the match.

  • All controllers are now supported (before was just Xbox One). Make sure to plug it in before launching or else it won't work. Also make sure to bind buttons appropriately in the Control menu before use. Note that as part of this change, menu up/down/left/right is no longer distinct from movement up/down/left/right, so rebind appropriately.

  • Changed some of Zero's animations to better, smoother ones.

  • Zero's Rakuhouha meter now will be grayed out if he doesn't have enough ammo to use it.

Balance Changes

  • Rakuhouha now grants Zero invulnerability while using it.

    • Commentary: This change makes Zero more relevant in late game despite not getting health upgrades and helps with his survivability. It also makes the move more strategic as it can be used both offensively and defensively. Plus, this is how the move worked in MMX4 so it's also more authentic to the real game.

  • Zero's special techniques no longer reflect projectiles.

    • Commentary: This wasn't really intentional and made it too easy to accidentally reflect X's shots (or, from an X player, randomly get screwed over from bad timing).

  • Guard and Rolling Shield now protect against freeze status.

    • Commentary: This is a quality of life change to make Zero's Hyouretsuzan down special less annoying and more counterable and make the game more strategic and interesting.

  • Cooldown added to Zero's dash attack.

    • Commentary: This makes the attack more balanced as it was kind of ridiculous when spammed.

  • X can no longer choose 2 or more of the same weapon.

    • Commentary: Choosing more than one of the same weapon can cause balance issues for weapons whose main balance aspect is their limited ammo (for example, Rolling Shield, Chameleon Sting, X-Buster.)

  • Uncharged Storm Tornado now costs 2 ammo.

    • Commentary: This helps the move become less spammable because using it repeatedly can get boring and frustrating to play against.
