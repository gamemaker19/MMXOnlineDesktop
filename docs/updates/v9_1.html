Version 9.1

Bug Fixes

  • Fixed a bug where Ray Splasher turrets couldn't be destroyed with certain weapons due to hit cooldowns not expiring.
  • Attempted to correct an issue in the acid.frag shader that was broken on some GPUs. In addition, the "failed to compile shaders" popup message now explicitly lists all shaders that failed to compile, to allow for easier troubleshooting for specific users experiencing shader issues.
  • Attempted fix for charged Storm Tornado bug that could cause crashes/black screens. (I cannot repro this locally, hence not 100% certain if it would be fixed.)
  • Fixed a bug where Hyper Buster was using the Kangaroo mech punch killfeed icon.
  • Added a collision wall on left side of Giant Dam map to prevent going out-of-bounds.

Balance Changes

  • Charged Tornado Fang: removed stun, increased damage from 1 to 2, and reduced damage speed by 25%.
  • X3 chest chip enhancement no longer provides invincibility, instead reducing damage by 75% while the barrier is active.
  • Reduced amount of ceiling spikes on Doppler Lab stage.
