Version 8.1

Changes

  • Fixed a bug that removed error resiliency in both server and client code, causing unnecessary crashes for both servers and game clients.
  • Allowed adding bots to specific teams in private matches.
  • Shippuuga got a new weapon id and killfeed index. There's a chance this might also fix issues where it doesn't flinch, but this is not certain.
  • Autobalance will no longer always target the host, but will select a random player instead.
  • Fixed a bug where a Giga Armor X hooked with charged Strike Chain showed with default armor
  • Fixed a bug where Item Tracer could give Zero Rakuhouha charge and "ding" invincible players.
