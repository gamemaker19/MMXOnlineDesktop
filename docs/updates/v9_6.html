Version 9.6

Note: these changes were reverted in 9.7 as a hotfix.

Improvements

  • Added a setting in Settings menu, "Disable shaders", to disable shaders. This can fix black square issues appearing on X's sprites and other graphical issues for certain users, and also slightly improve FPS, though disabling shaders means no special effects or weapon/hyper mode palette changes.
  • Added a setting in Settings menu, "Axl separate aim down & crouch". Only applies for Directional and Angular aim modes. Set this to "yes" if your keybind for Aim Down is different from the "Down" binding used by all characters for crouching. This will fix a corner case where if you aim down, you could uncrouch.
  • Updated some outdated and incorrect information.

Bug Fixes

  • Fixed a bug where Axl's copy shot sometimes wouldn't transform into the correct character/player.
  • Fixed a bug where wrong weapon slot could be selected as Axl in certain situations.
  • Fixed a bug where Axl cursor could randomly show up in top left corner of screen.
  • Fixed issues with certain maps where X could get under the stage, especially water maps with Bubble Splash.
  • Fixed a bug where Hadouken kill feed icon was using X Buster icon.
  • Fixed a bug where X could equip Golden Armor in death animation.
  • Attempted a blind bug fix where sometimes Hadouken and Shoryuken could cost 6 scrap. (Cannot repro locally.)
  • Fixed shader compile issues on some GPU architectures.
