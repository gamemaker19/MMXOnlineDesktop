Version 11.5

Improvements/bug fixes

  • Reduced bandwidth consumed per match by optimizing and compressing game data sent over the network.
  • Attempted a fix for a bug where a region's server could randomly crash.
  • Game clients prior to v11.4 should now properly display an update message.
  • Bots are now removed when the host leaves.
  • Removed ability for matches to be renamed after a match ends, as well as for offline matches.
  • Can now select a random map after a match ends by pressing the Menu Secondary binding when the cursor is on the map select.

Balance changes

Mega Man X

  • Chameleon Sting now can hit a max of two times, dealing up to 4 damage max.
