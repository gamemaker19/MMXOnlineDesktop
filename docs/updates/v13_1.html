Version 13.1

Changes

  • Fixed a bug where matches would only load for the preferred region configured in Settings.
  • Fixed a bug where some of Zero's technique slots could not be properly configured.
  • Fixed a bug where Zero could equip two of the same technique at once.
  • Fixed a bug where Buster could not properly be equipped in X's second or third slot.
  • Fixed Longshot Gizmo weapon switch bug by preventing weapon switch while shooting a gizmo volley.
  • Fixed a bug where Axl couldn't charge Assassination when disguised as Vile.
  • Fixed a bug where Longshot Gizmo (and other weapons) could be used while reviving into Vile MK-II.
  • Fixed a bug where Zero's giga attack icon for the opponent could be incorrectly displayed in the HUD.
  • Because the low quality text option was causing some issues for some users, it is now off by default, even in low quality presets, and has to manually be set to "Low".
